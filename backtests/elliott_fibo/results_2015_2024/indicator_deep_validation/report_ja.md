# 有力インジケータ 本格検証

## 結論サマリー

- `T5 + MACDヒストグラム上昇` は利益の上積みが最も大きい攻め候補。
- `T4 + BB位置 + 終値位置` はバランス型。通貨別・後半期間の安定性が良い。
- `ADXは高いほど良い` ではなく、今回の構造では `ADX<25` のような上限フィルタが有効。
- `BB位置` と `終値位置` は閾値を少し動かしても大崩れしづらく、Pine化の優先度が高い。
