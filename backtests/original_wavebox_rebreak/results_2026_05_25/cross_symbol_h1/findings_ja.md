## 実践化への判断

WaveBox H1 Rebreak は、全通貨共通手法ではなく、銘柄別運用にする。

現時点の候補:

- 主力: USDJPY H1
- 追加研究: GBPJPY long only / XAUUSD / EURJPY short only
- 除外: AUDJPY / CHFJPY / SILVER long

USDJPYの実践候補:

- H1
- box 8本
- box幅 2.0ATR以内
- TP 1.5R
- H4逆行なしを推奨
- A級は戻し50%-61.8%
- body ratio 0.35-0.45の死にゾーンは除外候補

## 課題

USDJPY内では残るが、2019-2022に弱い。
特に2019/2021/2022は年単位で負けている。

したがって次は、負け年に共通する環境を見つける必要がある。

候補:

- H4/D1のレンジ環境
- 低ボラ・高ボラの極端環境
- 方向別ではショートが弱い年がある
- 時間帯では 1時、6時、14時が弱い
- 月別では8月が弱い

次の改善テーマ:

1. USDJPY専用Pineを作る
2. ただし実運用前に、H4/D1レンジ回避フィルタを検証する
3. 時間帯 1時/6時/14時を除外するか検証する
4. ロング優先、ショートは追加条件付きにするか検証する
