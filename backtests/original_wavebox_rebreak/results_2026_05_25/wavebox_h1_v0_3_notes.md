# WaveBox USDJPY H1 Rebreak v0.3

作成日: 2026-05-25

## v0.3の目的

v0.2は `H1 / box8 / 2.0ATR / TP 1.5R` を基礎にした。
v0.3では、弱い時間帯と深すぎる戻しを削って、年別の崩れを小さくする。

## v0.3コアルール

- 対象: USDJPY
- 時間足: H1
- H4は表示するが、コアルールでは必須にしない
- 1波2波をpivotで認識
- 2波戻し: 50%-76.4%
- P2確定後、直近8本box
- box幅: 2.0ATR以内
- boxを3波方向へ終値ブレイク
- EMA20が同方向
- TP: 1.5R
- SL: `P2/box外側 + 0.2ATR`
- 除外時間: 1時 / 6時 / 11時 / 14時

時間は元データのブローカー時刻基準。
TradingView/Pineでは時差確認が必要。
