# H4/D1 優位性候補の人間再現プラン

このメモは、初回検証で良かった次の3条件を、人間がTradingViewなどのチャート上で再現確認できる形に翻訳したものです。

- H4 `V字＋高値停滞`
- H4 `レンジ抜け2回目以降のブレイクアウト`
- D1 `値幅広めの高値停滞`

注意: ここでの「完全再現」は、現在の機械検証ロジックの条件を再現するという意味です。Sai氏本人の裁量判断を完全に再現するものではありません。

## 共通ルール

### 対象

- 時間足: H4 または D1
- 通貨: 初回検証では `AUDJPY / CHFJPY / EURJPY / GBPJPY / XAGUSD / XAUUSD`
- USDJPYは除外
- 12月15日から1月10日は除外

### エントリー方向

- `高値停滞` はロングのみ
- `レンジ抜け2回目以降` はロングとショート両方あり

### エントリー方式

1. 条件確定はローソク足の終値で判断
2. エントリーは次の足の始値
3. ATRは14本ATR
4. ブレイク判定は `0.1 ATR` 以上の余白を要求

### 中短一致フィルタ

エントリー前に、必ず中期方向と短期方向が同じであること。

H4の場合:

- 中期: 約130本、約21日分
- 短期: 直近2本から18本程度
- モメンタム: 直近3本前後で、実体の過半が同方向、かつ値幅が1ATR以上

D1の場合:

- 中期: 約22本、約1ヶ月分
- 短期: 直近2本から3本程度
- モメンタム: 直近2本前後で、実体の過半が同方向、かつ値幅が1ATR以上

人間確認では、次のように見る。

- ロング: 中期で高値/安値が切り上がる、短期も上向き、直近の陽線実体が強い
- ショート: 中期で高値/安値が切り下がる、短期も下向き、直近の陰線実体が強い

## 1. H4 V字＋高値停滞

初回検証:

- 総R: +27.68R
- 勝率: 53.57%
- 取引数: 28件

### コード上の定義

H4で次の条件がすべてそろう。

1. 中期も短期もロング方向
2. 直近に上方向モメンタムがある
3. 過去24本、約4日以内にV字回復がある
4. V字の谷は、24本窓の端ではなく中央寄りにある
5. 谷までの下落幅が3ATR以上
6. 谷からの回復が下落幅の80%以上
7. 高値停滞がある
8. 停滞は直近2本から6本程度
9. 停滞ゾーンの高値が、過去180本、約30日高値付近にある
10. 確定足終値が停滞ゾーン高値を `0.1 ATR` 以上上抜ける

### 人間用チェックリスト

次の順に見る。

1. H4チャートを約1ヶ月表示する
2. まず中期が上昇トレンドか確認する
3. 直近4日以内に急落から急回復したV字があるか見る
4. V字の戻りが、下落幅の8割以上戻しているか確認する
5. V字回復後、過去1ヶ月高値付近で2から6本ほど横ばいしているか見る
6. 横ばいの上限を終値で明確に上抜けたら候補
7. 次足始値でロング
8. SLは停滞ゾーン下限の少し下、目安は `ゾーン下限 - 0.25ATR`

### 見送り条件

- V字ではなく、だらだら戻している
- V字はあるが、過去1ヶ月高値付近ではない
- 停滞が1本だけ
- 停滞幅が大きすぎて、ただの荒れた相場に見える
- 中期上昇ではなく、レンジ中央にいる

## 2. H4 レンジ抜け2回目以降のブレイクアウト

初回検証:

- 総R: +27.68R
- 勝率: 40.82%
- 取引数: 98件

### コード上の定義

H4で次の条件がそろう。

1. 中期も短期も同方向
2. 同方向のモメンタムがある
3. 約20日分、120本程度のレンジがある
4. そのレンジの値幅が10ATR以内
5. レンジ後の約40本、約6.5日以内に1回目のブレイクがある
6. 1回目のブレイク後、終値がレンジ内へ戻る
7. その後、1回目ブレイクの最高値/最安値をさらに `0.1 ATR` 以上抜く

ロングの場合:

- レンジ上限を1回上抜ける
- 終値でレンジ内へ戻る
- その後、1回目ブレイク高値をさらに上抜ける

ショートの場合:

- レンジ下限を1回下抜ける
- 終値でレンジ内へ戻る
- その後、1回目ブレイク安値をさらに下抜ける

### 人間用チェックリスト

1. H4で約1ヶ月分を表示する
2. 20日前後のレンジを探す
3. レンジ上限/下限を水平線で引く
4. 1回目のブレイクは見送る
5. ブレイク後、終値でレンジ内へ戻ったことを確認する
6. 再度ブレイクし、1回目のブレイク極値を超えたら候補
7. 次足始値でエントリー
8. SLはロングなら1回目ブレイク高値から1ATR下、ショートなら1回目ブレイク安値から1ATR上

### 見送り条件

- そもそもレンジがない
- 1回目のブレイクで戻らず、そのまま伸びた
- 2回目に見えるが、ただの上昇トレンド中の高値更新
- レンジ幅が広すぎる
- 中短一致がない

## 3. D1 値幅広めの高値停滞

初回検証:

- 総R: +14.89R
- 勝率: 58.49%
- 取引数: 53件

### コード上の定義

D1で次の条件がそろう。

1. 中期も短期もロング方向
2. 上方向のモメンタムがある
3. 直近2本から3本程度で高値圏に停滞している
4. 停滞幅は通常停滞より広い
5. 具体的には、停滞ゾーン幅が `1.2 ATR` より大きく、`2.5 ATR` 以内
6. 確定足終値が停滞ゾーン高値を `0.1 ATR` 以上上抜ける
7. 過去30本、約1ヶ月高値付近ならより良い

### 人間用チェックリスト

1. D1チャートで1ヶ月から3ヶ月を表示する
2. 中期上昇トレンドか確認する
3. 直近数日で高値圏にいるか確認する
4. 2から3日ほど横ばいだが、普通のタイトな停滞より上下幅があるか見る
5. その横ばい上限を終値で明確に上抜ける
6. 次の日の始値でロング
7. SLは停滞ゾーン下限の少し下、目安は `ゾーン下限 - 0.25ATR`

### 見送り条件

- 停滞が1日だけ
- 停滞幅が2.5ATRを超える
- 高値圏ではなくレンジ中央
- 日足の中期方向が上向きではない
- 上抜けがヒゲだけで終値確定していない

## 実践検証プラン

### Step 1: 過去チャートで目視マーキング

TradingViewで対象通貨を開き、H4/D1それぞれで以下を手作業で記録する。

- 日付
- 通貨
- 時間足
- 手法名
- エントリー方向
- エントリー価格
- SL
- 利確/決済候補
- その場で見えた根拠
- 見送り理由があれば記録

### Step 2: 合格/不合格を分ける

各候補を次の3段階で分類する。

- A: ほぼ機械条件どおりで、人間の目でもきれい
- B: 条件は満たすが形が微妙
- C: 機械的には拾うが、人間なら見送りたい

最初にPine化するのはAだけでよい。

### Step 3: Pineで可視化する条件

最初のPineでは、売買まではせず、次だけを表示する。

- H4 `V字＋高値停滞` 候補ラベル
- H4 `レンジ抜け2回目以降` 候補ラベル
- D1 `値幅広めの高値停滞` 候補ラベル
- 停滞ゾーンのボックス
- レンジ上限/下限
- 1回目ブレイクの極値
- 2回目ブレイク確定足

### Step 4: 最終ルール化

Pineで目視確認したあと、次のように絞る。

- 勝てているが形が汚いものを除外
- 勝率よりも、損切りが浅くできる形を優先
- XAUUSDとクロス円で条件を分ける
- H4/D1は取引数が少ないため、最低でも20から30件の目視サンプルを確認する

## 最初に狙う優先順位

1. H4 `V字＋高値停滞`
2. H4 `レンジ抜け2回目以降のブレイクアウト`
3. D1 `値幅広めの高値停滞`

理由:

- H4 `V字＋高値停滞` は勝率とPFが最も見やすい
- H4 `レンジ抜け2回目以降` は取引数が多く、検証材料が多い
- D1 `値幅広めの高値停滞` は勝率は高いが、取引数が少なく、保有期間が長い
