# Clean H4 V Reclaim Pine実装メモ

作成日: 2026-05-29

## 目的

`D1売り否定 + H4 V右肩リクレイム` 検証で最も実戦向きだった候補を、TradingView上で確認できるPineストラテジーに落とし込む。

Pine対象:

- `AVOID_D20_OR_RSI_30D + RS120_BODY45_CLOSE60_RR1.5`
- XAUUSD/XAUは初期設定で除外
- H4チャートでの確認を前提

## 実装ファイル

- `pine/research/clean_h4_v_reclaim_strategy.pine`

## ルール

### H4エントリー

1. confirmed pivot high -> confirmed pivot low の急落V
2. 下落幅 >= 3.2ATR
3. 左肩速度 >= 0.25ATR/本
4. 右肩速度 >= 左肩速度 x 1.20
5. 右肩終値が左肩起点 + 0.05ATR を上抜け
6. シグナル足の実体比率 >= 45%
7. シグナル足の終値位置 >= 60%
8. V安値確定後に、V安値 - 0.10ATR を割っていない

### D1見送りフィルタ

直近30日以内に以下のどちらかが出ていたら、新規エントリーしない。

- D1 Donchian20下抜け否定_Q
- D1 RSI30割れ否定_Q

D1の否定は日足が確定しないと分からないため、Pine側では否定日の翌日から見送り期間を開始する。

### 売買

- Entry: 次H4足始値
- SL: V安値 - 0.25ATR
- TP: シグナル足終値基準の1.5R
- 最大保有: H4 180本

## Pine確認手順

1. TradingViewでH4チャートに適用する
2. まず `XAUUSD/XAUは除外` をONのまま確認する
3. 状態テーブルで `D1 avoid` が `OFF 許可` の時だけシグナルが出ることを確認する
4. `見送り候補を表示` をONにして、D1フィルタ・右肩速度不足・足品質不足で消えている候補を確認する
5. Pythonとの照合は、最初に成績ではなく `V起点 / V安値 / シグナル足 / D1 avoid状態` の一致を見る

## 注意

TradingViewのストラテジーテスター成績はPythonと完全一致しない可能性がある。

主な理由:

- TradingViewとPythonでデータ提供元・タイムゾーン・欠損足が違う可能性
- Pythonはスプレッド/スリッページをR換算で処理している
- Pineは同一足でSL/TPに触れた場合の処理がTradingView仕様になる
- TPはPython検証に合わせてシグナル足終値基準にしているため、次足始値のギャップが大きい場面では実効RRが少しずれる

したがって、最初の目的は「完全な損益一致」ではなく「シグナル位置の一致」とする。
