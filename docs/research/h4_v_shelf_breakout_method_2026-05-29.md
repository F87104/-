# H4 V Shelf Breakout 新手法メモ

作成日: 2026-05-29

## 手法名

**H4 V Shelf Breakout**

日本語名: **H4 V字否定後の棚ブレイク**

## 発想

写真の共通点は、V字そのものではなく、**急落が否定されたあとに、上側で価格が崩れずに棚を作っていること**。

つまり買うべき場所は谷ではない。急落後の最初の反発でもない。

買う場所は、以下の順番がそろった後。

1. 上昇地合い、または上昇へ転換し始めた地合い
2. H4で鋭い急落
3. 急落がすぐ否定され、下落幅の中腹以上まで回復
4. 回復後、価格が上で崩れずに横ばいの棚を作る
5. その棚の高値を終値で抜ける

この手法は、V字の底を当てる逆張りではなく、**売りの失敗を確認した後の上昇再開を買う順張り寄りの手法**。

## 写真から見えた共通パターン

| 要素 | 内容 |
|---|---|
| 地合い | 完全な下落中ではなく、上昇中の押し目、または底打ち後の回復局面 |
| 急落 | H4で短期間に鋭く下げる |
| 否定 | 急落後、すぐに中腹以上まで戻す |
| 棚 | 戻った後に数本から十数本、上側で横ばいになる |
| Entry | 棚の高値を終値で抜ける |
| SL | V谷ではなく棚安値を基本にする |
| TP | まずは2R。上位高値が近い場合は利確候補にする |

## 既存T5との違い

| 項目 | H4 V候補 T5 | H4 V Shelf Breakout |
|---|---|---|
| Vの扱い | 61.8-80%回復候補 | 急落否定後の棚形成が主役 |
| Entry | 高値停滞 or 再ブレイク | 棚の高値終値ブレイク |
| 重要条件 | MACD/BB一致 | 棚の圧縮・保持・リスク幅 |
| SL | V谷ベース | 棚安値ベースを基本 |
| 狙い | V回復後の再上昇 | 急落否定後の上昇再開 |

## 初期ルール案

### 環境

- 時間足: H4
- 方向: ロングのみ
- 地合い: H4 EMA20 > EMA50 または D1 EMA20 > EMA50
- 12/15から1/10は新規停止

### Vショック条件

- confirmed pivot high -> confirmed pivot low
- 急落幅: 2.8ATR以上
- 急落速度: 0.25ATR/本以上
- 急落本数: 2から24本
- 回復率: 65%から110%
- 回復本数 / 下落本数: 1.8倍以内
- 谷を0.2ATR以上割ったら失効

### 棚条件

- 回復後、H4で5本以上の棚を見る
- 棚の値幅: 1.25ATR以内
- 棚の安値が急落幅の50%回復ラインを維持
- 棚高値を終値で0.05ATR以上抜ける
- ブレイク足の実体比率: 45%以上
- ブレイク足の終値位置: 足の上側40%以内

### リスク

- SL: 棚安値 - 0.25ATR
- 最大リスク幅: 2.2ATR以内
- TP: Entry基準で2R
- 最大保有: H4 120本

## 見送り理由

以下は見送る。

- H4以外で検証している
- 地合いが上向きでない
- 棚が広すぎる
- 棚の安値が急落中腹を維持できていない
- ブレイク足の実体または終値位置が弱い
- SLまでの距離が広すぎる
- EMA20から離れすぎている
- すでにポジション保有中
- 候補発生から時間が経ちすぎた

## 仮説

この手法の本質は、V字ではなく **急落否定後の上側滞在時間**。

急落後にすぐ戻しただけなら、まだただのショートカバーかもしれない。
しかし、戻した後に上側で崩れず棚を作るなら、売りが失敗し、買いが主導権を取り返した可能性が高い。

その棚を抜ける瞬間だけを狙う。
