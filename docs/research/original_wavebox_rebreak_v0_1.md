# Original Method v0.1: WaveBox Rebreak

作成日: 2026-05-25

Synapse検証から見えた弱点を避けるため、手法そのものを作り替える。
目的は「三尊/逆三尊を当てること」ではなく、「2波終端の圧縮から3波再開だけを拾うこと」。

## 1. なぜ変えるか

Synapse定義グリッドの初回検証で分かったこと:

- M5はノイズが多すぎる
- B単独ブレイクは弱い
- 斜め抜け + B抜けは比較的良い
- H1/M15/H4の波認識は数字が残る
- 厳格な三尊/逆三尊に寄せすぎると候補が消える
- 半値TPは裁量目標として良いが、機械出口としては不安定

そこで、形を「三尊/逆三尊」から「押し戻し後の圧縮ボックス再ブレイク」に変える。

## 2. 手法の核

WaveBox Rebreak は、次の順番で見る。

1. H1/M30/M15/H4で、明確な1波と2波を探す
2. 2波の戻しが 50%-88.6% に入ることを確認する
3. 2波終端後、価格が小さなボックスに圧縮されるのを待つ
4. ボックスを3波方向へ再ブレイクしたら候補
5. 上位足が逆行していない候補だけをA級にする

つまり、A/Bを無理に三尊の肩から取らない。
Bは「圧縮ボックスの外側」として再定義する。

## 3. ロング定義

ロングは、上昇1波の後の押し目から3波上昇を狙う。

中波:

- `P0`: 上昇1波の起点安値
- `P1`: 上昇1波の終点高値
- `P2`: 2波押しの終点候補安値

条件:

- `P0 -> P1` の値幅が十分ある
- `P2` が `P0` を明確に割っていない
- 戻し率 `(P1 - P2) / (P1 - P0)` が 0.50-0.886
- 2波の横軸が短すぎない
- `P2` 確定後、数本の小レンジボックスができる
- 終値がボックス上限を上抜く
- ブレイク時点で短期EMAも上向き
- Bから離れすぎていない

SL:

- `min(P2, boxLow) - 0.2ATR`

TP:

- 固定 1.5R
- 固定 2R
- 半値TPは参考線としてのみ使う

## 4. ショート定義

ショートは上下を反転する。

中波:

- `P0`: 下降1波の起点高値
- `P1`: 下降1波の終点安値
- `P2`: 2波戻りの終点候補高値

条件:

- `P0 -> P1` の値幅が十分ある
- `P2` が `P0` を明確に超えていない
- 戻し率 `(P2 - P1) / (P0 - P1)` が 0.50-0.886
- `P2` 確定後、数本の小レンジボックスができる
- 終値がボックス下限を下抜く
- ブレイク時点で短期EMAも下向き
- Bから離れすぎていない

SL:

- `max(P2, boxHigh) + 0.2ATR`

## 5. 旧Synapseとの違い

旧Synapse:

- 小波の三尊/逆三尊を探す
- A/Bを波形の役割線として決める
- B抜けと斜め抜けで反転確認する

WaveBox Rebreak:

- 三尊/逆三尊を必須にしない
- Aは使わない
- Bは圧縮ボックスの外側
- 斜めラインの代わりに、2波終端後の圧縮と短期EMA反転を見る
- エントリーは「形」ではなく「押し戻し後の再始動」に置く

## 6. 実践ランク

A級:

- H1/H4/M30で成立
- 戻し 0.50-0.886
- 上位足1つ以上順行
- 明確な逆行なし
- ボックス幅がATRの1.2-1.6倍以内
- 実体比率0.35以上
- 予定RR 1.5以上

B級:

- M15で成立
- 上位足は中立以上
- ボックス幅はATRの2.0倍以内
- 予定RR 1.5以上

観察:

- M5
- 上位足逆行あり
- 戻しが浅い
- すでにP1方向へ回復しすぎている

## 7. 初回検証で見る指標

- 時間足別: M15/M30/H1/H4
- box bars: 6 / 8 / 12
- box width: 1.2ATR / 1.6ATR / 2.0ATR
- filter: base / context / quality
- TP: 1.5R / 2R
- 勝率
- 平均R
- PF
- 最大DD
- OOS 2025-2026

合格ライン:

- 全期間PF 1.10以上
- OOS PF 1.05以上
- 平均R 0.05以上
- 件数が少なすぎない
- M5依存でない
