# USDJPY H1 WaveBox Rebreak v1 実戦仕様

作成日: 2026-05-26

## 結論

USDJPY H1 WaveBox Rebreak は、現時点では `USDJPY / H1` 専用の実戦候補。
万能手法ではなく、「強い1波のあと、2波押しが止まり、8本ボックスを再ブレイクするところだけ」を狙う。

実戦前提条件:

- `/Users/asamifujita/Documents/Codex/2026-05-21/fx-ai/docs/wavebox_operational_preconditions_v1.md`

実戦標準は `v0.4 filtered + 1波剪定` とする。

- 戻し: 50%-78.6%
- 1波値幅: 3.0ATR以上
- 1波速度: 0.35ATR/本以上
- 1波時間: 24本以内
- 除外時間: entry hour 1 / 6 / 14
- Box: 直近8本
- Box幅: 2.0ATR以内
- TP: 1.5R
- SL: P2またはBox外側 + 0.2ATR
- 最大保有: 72本

## 何の手法か

これはV字の底を当てる手法ではない。

ロングなら、

1. H1で強い上昇1波が出る
2. その後に50%-78.6%まで押す
3. 押し目の終端付近で下げ止まり、8本程度の停滞Boxを作る
4. Box上限を終値で抜く
5. 次のH1始値で買う

ショートは上下反転。

狙っているのは「落ちている最中」ではなく、押し目が止まってからの再上昇。
つまり `押し目買い + 停滞確認 + 再ブレイク`。

## 正式条件

### ロング

- `P0`: 1波起点の安値
- `P1`: 1波終点の高値
- `P2`: 2波押しの終点候補安値
- Pivot幅: 3
- Pivot間の最小スイング: 1.15ATR
- 1波値幅: 3.0ATR以上
- 1波速度: 0.35ATR/本以上
- 1波時間: 24本以内
- P2はP0を明確に割らない
- 戻し率: `(P1 - P2) / (P1 - P0)` が 0.50-0.786
- P2確定後36本以内
- 直近8本のBox幅が2.0ATR以内
- 終値がBox上限 + 0.05ATRを上抜く
- 前足終値はまだBox上限を上抜いていない
- EMA20より上、EMA20傾き上向き
- ブレイクの追いかけ幅が0.80ATR以内
- ブレイク足の実体率が0.25以上
- エントリー時刻が1時/6時/14時ではない
- 12/15以降と1/10以前は除外

### ショート

ロング条件を上下反転。

- `P0`: 1波起点の高値
- `P1`: 1波終点の安値
- `P2`: 2波戻りの終点候補高値
- 戻し率: `(P2 - P1) / (P0 - P1)` が 0.50-0.786
- 終値がBox下限 - 0.05ATRを下抜く
- EMA20より下、EMA20傾き下向き

## ランク

### B

標準条件を満たす。
実戦では最小ロット、またはフォワード観察。

### A

標準条件に加えて、H4が逆行していない。
ただし今回の検証では、H4条件だけで劇的に成績が上がるわけではない。
ロットを上げる根拠というより、心理的に扱いやすい候補。

### A+

標準条件に加えて、戻しが50%-61.8%。
今回の検証で最も質が高いフィルター。

H4逆行なしよりも、浅い押しの方が優先。

## 実運用ルール

- まずはUSDJPY H1のみ。
- Pineのプリセットは `v0.4 filtered`。
- `1波剪定` はON、プリセットは `Strict` を標準にする。
- 回数を増やす検証では、まず `Balanced` を使う。いきなり `Loose` やOFFへ落とさない。
- Action分類は `GO A+` / `GO A` を実戦候補、`SELECT A` を裁量確認、`OBS B` を観察または小ロット、`SKIP` を見送りにする。
- レンジ抜け確認は研究オプション。標準ではOFF。
- TradingViewの除外時間タイムゾーンは、検証CSVに合わせて確認する。
- SIGNALが出た足では入らない。次のH1始値がENTRY。
- TPは1.5R固定。
- SLはP2/Box外側 + 0.2ATR。
- 最大保有72本で時間決済。
- フォワード20-30件までは1回リスク0.25%-0.5%上限。
- 連敗4回、または月間-3Rで停止して目視レビュー。
- A+だけロットを上げる判断は、フォワード後に行う。

## 見送り条件

- USDJPY H1以外
- 戻しが50%未満
- 戻しが78.6%を超える
- Box化せず一気に伸びている
- Boxブレイク時にすでに0.80ATR以上追いかけている
- 1時/6時/14時エントリー
- 年末年始
- 指標直前など、スプレッドが通常より広い場面
