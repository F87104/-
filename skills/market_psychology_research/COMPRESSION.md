# Compression — 圧縮

> 低ボラ・停滞の中でポジションが静かに蓄積され、
> 均衡が破れた瞬間に一方向へ爆発する構造を研究する。

[← SKILL.md に戻る](SKILL.md)

---

## 1. 概要

Compression（圧縮）とは、**ボラティリティが低下した停滞局面でポジションが静かに蓄積され、
均衡が破れた瞬間に一方向へ爆発的に動く構造** である。

低ボラ・レンジ相場では、相場が動かないことに対する「油断」が生まれる。
売りと買いが拮抗し、両側のポジションが狭いレンジに密集していく。
このレンジが破れると、逆側のポジションが一斉に損切りを迫られ、
蓄積されたエネルギーが一方向に解放される。

圧縮そのものは方向を持たない。重要なのは **「いつ・どちらに破れるか」** であり、
ブレイク後はしばしば [Short Squeeze](SHORT_SQUEEZE.md) / [Long Liquidation](LONG_LIQUIDATION.md) /
[FOMO](FOMO.md) へと遷移する。

## 2. 市場参加者の心理

- **期待していた人**: 「レンジは続く」と考える逆張り・両建て・オプション売りの参加者。
- **失望の瞬間**: レンジが破れ、想定外の方向に動き出す。
- **損切りする人**: レンジ継続を前提にポジションを取っていた参加者。ブレイク方向と逆側が投げる。
- **加速の方向**: ブレイクした方向。蓄積された逆側ポジションの投げが燃料になる。
- **油断していた人**: 「動かないから安全」と気を緩めた参加者ほど、爆発時の被害が大きい。

> 核心: **動かないことそのものが、次の大きな動きのエネルギーを溜めている。**

---

## 3. チャート上の構造

```
低ボラ（ATR低下・値幅縮小）
  ↓
停滞（レンジ・横ばいの継続）
  ↓
ポジション蓄積（両側の建玉がレンジ内に密集）
  ↓
爆発（レンジブレイク → 一方向への急走）
```

- ボリンジャーバンドが収縮（スクイーズ）し、バンド幅が極小になる。
- ATR が低下を続け、値幅が縮小する。
- レンジが長く続くほど、ブレイク時のエネルギーが大きくなる傾向。

---

## 4. 数値化できる特徴

| 特徴量 | 観測の狙い |
|---|---|
| **ATR低下** | ATR が直近平均を下回り続けているか（圧縮の深さ） |
| **レンジ継続** | 一定レンジ内に収まったバー数（圧縮の期間） |
| **ボリンジャー収縮** | バンド幅 (`upper - lower`) が直近最小か（スクイーズ） |
| **値幅縮小** | 高安レンジが縮小傾向か |
| **ブレイク方向** | レンジ上下どちらを終値で破ったか |

---

## 5. 検証仮説

- **H1**: ボリンジャーバンド幅が直近 N バーの最小水準まで収縮した後にレンジをブレイクすると、
  ブレイク後 M バーでボラ拡大と一方向の加速が有意に観測される。
- **H2**: 圧縮期間が長いほど、ブレイク後の値幅が大きい（蓄積エネルギー仮説）。
- **H3**: ATR低下とバンド収縮が同時に成立した圧縮は、単一条件より OOS 再現性が高い。

---

## 6. Python実装候補

- ボリンジャーバンド幅: `(upper - lower) / middle` を算出し、直近最小かを判定。
- ATR低下フラグ: `atr < atr.rolling(N).mean()` の継続。
- レンジ判定: 直近の高安レンジが ATR の一定倍以下に収まる連続バー数。
- ブレイク検出: レンジ上下限の終値ブレイク。
- イベントスタディ: ブレイク後のボラ拡大・リターンを圧縮期間で層別して OOS 集計。

```python
# 方針（疑似コード）
def detect_compression_breakout(df, bb_window, squeeze_pctile, range_n):
    bw = bb_width(df, bb_window)
    squeezed = bw[-1] <= bw.rolling(range_n).quantile(squeeze_pctile)[-1]
    rng = recent_range(df, range_n)
    breakout = df.close[-1] > rng.high or df.close[-1] < rng.low
    return squeezed and breakout
```

## 8. 今後の研究課題

- ブレイク **方向** の事前予測は可能か（圧縮自体は方向中立）。
- だましブレイク（[Trap](TRAP.md)）と本物のブレイクの分離。
- 圧縮の入れ子構造（上位足の圧縮の中に下位足の圧縮）。
- 圧縮 → 爆発 → [FOMO](FOMO.md) という遷移の連鎖を定式化する。
