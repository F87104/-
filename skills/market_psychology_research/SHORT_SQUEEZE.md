# Short Squeeze — 売り方の踏み上げ

> 急落で増えた新規売りが、価格が下に伸びないことで買い戻しを迫られ、
> その買い戻しが上昇を加速させる構造を研究する。

[← SKILL.md に戻る](SKILL.md)

---

## 1. 概要

Short Squeeze（踏み上げ）とは、**売りポジションの損切り（買い戻し）が、上昇の燃料になる構造** である。

急落が起きると「もっと下がる」という続落期待のもとで新規の売りが積み上がる。
ところが価格が下に伸びなくなると、売り方は含み損に転じ、買い戻しを迫られる。
売りの買い戻しは「買い注文」であるため、価格は上昇する。
上昇すると更に他の売り方が踏まれ、買い戻しの連鎖（squeeze）が起きて上昇が加速する。

これは [Expectation Failure](EXPECTATION_FAILURE.md)（続落期待の崩壊）から
直接派生する代表的な加速構造である。

---

## 2. 市場参加者の心理

- **期待していた人**: 急落に乗って新規売りを入れた参加者。「続落」を期待。
- **失望の瞬間**: 安値を更新できず、価格が下に伸びない。
- **損切りする人**: 直近で売った参加者。含み損が拡大し、買い戻し（損切り）を迫られる。
- **加速の方向**: **上方向**。買い戻しが買い圧力となり、踏み上げの連鎖を生む。
- **油断していた人**: 「戻りは売り」と機械的に構えていた参加者ほど、連鎖の燃料になる。

> 核心: **下に伸びないこと自体が、上昇のエネルギー源になる。**

---

## 3. チャート上の構造

```
急落
  ↓
新規売り増加（続落期待）
  ↓
下に伸びない（安値更新の停止）
  ↓
買い戻し（売り方の損切り）
  ↓
加速（踏み上げの連鎖で上昇加速）
```

- 急落の直後、安値圏で **棚（shelf, 横ばいの停滞）** が形成されることが多い。
- 棚は「売り方が下を試したが伸びなかった」記録であり、ここに売りの含み損が溜まる。
- 棚を上にブレイクした瞬間、買い戻しが一斉に発生して加速する。

## 4. 数値化できる特徴

| 特徴量 | 観測の狙い |
|---|---|
| **V字回復** | 急落幅に対する戻り幅の比率。踏み上げの初動を測る |
| **棚形成 (shelf)** | 安値圏での横ばい区間の長さとレンジ幅（ATR比） |
| **棚ブレイク** | 棚の上限を終値で突破したか。踏み上げ発火の合図 |
| **Donchian更新** | 直近 N バーの高値（Donchian 上限）を更新したか |
| **出来高** | 棚ブレイク時の出来高スパイク（買い戻しの集中度） |

## 5. 検証仮説

- **H1**: 急落後に安値圏で棚が形成され、その上限を終値ブレイクすると、
  ブレイク後 M バーで上方向の加速が有意に観測される。
- **H2**: 棚のレンジが狭い（ATR比が小さい）ほど、ブレイク後の加速が大きい（圧縮の効果）。
- **H3**: 棚ブレイクと同時に Donchian 上限を更新したケースは、OOS での再現性が高い。

## 6. Python実装候補

- 棚検出: 急落後の区間で、価格レンジが ATR の一定倍以下に収まる連続バーを「棚」と定義。
- 棚ブレイク検出: 棚上限を終値で上抜けたバーをイベントとして記録。
- Donchian 更新フラグ: `rolling(N).max()` を終値が上回ったか。
- イベントスタディ: 棚ブレイク後のリターン分布を OOS で集計し、棚の幅・長さでの層別を行う。

```python
# 方針（疑似コード）
def detect_short_squeeze(df, drop_lb, shelf_max_atr, donchian_n):
    drop = recent_sharp_drop(df, drop_lb)
    shelf = find_low_range_shelf(df, max_atr=shelf_max_atr)   # 安値圏の横ばい
    breakout = df.close[-1] > shelf.high
    donchian_break = df.close[-1] > df.high.rolling(donchian_n).max()[-2]
    return drop and shelf.valid and breakout and donchian_break
```

## 8. 今後の研究課題

- **事例研究**: **H4 V Initial Shelf Breakout** はこの構造の主要事例。
  急落 → V字 → 初期棚 → 棚ブレイクは、踏み上げの教科書的フローである。
- 棚の「質」を定義する（売りの含み損がどれだけ溜まっているかの代理指標）。
- [Compression](COMPRESSION.md) との接続: 棚 = 局所的な圧縮と捉え、爆発の方向を予測する。
- [Long Liquidation](LONG_LIQUIDATION.md) は本構造の完全対称モデル。両者の特徴量を共通化する。
